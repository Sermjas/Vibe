# 🧾 Receipt OCR Vibe-Bot

> **AI-ассистент для учета финансов**, созданный в парадигме **VibeCoding** с помощью [Cursor](https://cursor.com). Бот мгновенно извлекает сумму покупки из фотографий чеков, используя мощь Google Gemini.

---

## ✨ Основные возможности
* **Gemini 2.5 Flash Lite:** Использование LMM-модели последнего поколения для точного распознавания текста (OCR).
* **Умная обработка лимитов:** Автоматический `cooldown` на 10 минут при достижении квот API (Error 429) и экспоненциальная задержка при повторных попытках.
* **Smart Compression:** Встроенное сжатие изображений через `Pillow` для быстрой загрузки и экономии токенов.
* **Чистый результат:** Бот настроен возвращать только итоговую сумму (число), игнорируя лишний шум и текст.

## 🛠 Технологический стек
* **Core:** Python 3.10+
* **Framework:** `aiogram 3.x` (Asynchronous)
* **AI Engine:** `google-genai` (Gemini API)
* **Image Processing:** `Pillow`
* **Env Management:** `python-dotenv`
